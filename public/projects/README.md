# Project Cover Images

Tempatkan file gambar cover proyek Anda di folder ini.

## Aturan Penamaan:
Nama file gambar **WAJIB** sama dengan nama repositori GitHub Anda di-mapping ke lowercase dan dengan ekstensi `.jpg`.

### Contoh:
1. Jika nama repositori di GitHub adalah `smart-irrigation-system`, simpan cover-nya dengan nama `smart-irrigation-system.jpg`.
2. Jika nama repositori di GitHub adalah `eco-store-front`, simpan cover-nya dengan nama `eco-store-front.jpg`.

## Rekomendasi Dimensi:
- Aspek Rasio: 16:9
- Resolusi yang disarankan: `1280x720` piksel atau `640x360` piksel.
- Format: `.jpg` (untuk optimasi loading web).

*Catatan: Jika gambar cover tidak ditemukan, sistem otomatis memunculkan ilustrasi kode minimalis interaktif yang sangat keren sebagai fallback.*
